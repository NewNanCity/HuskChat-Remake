## 🆕 What's New in HuskChat Remake

HuskChat Remake extends the original HuskChat with:

- **🚀 Extended Event System** - Complete event API for all chat types
- **🛠️ Enhanced API Interface** - Unified message sending, channel management APIs
- **📱 Modern Architecture** - Optimized for latest Minecraft server platforms
- **🔧 Better Extensibility** - Plugin-based design supporting custom extensions
